# About me

I’m a backend-focused software engineering student (8th semester at UDESC) and junior developer with a deep interest in **software architecture** and cloud-native systems. I love exploring how systems evolve, scale, and communicate through modern design principles :)


---

## Interests
- Software architecture and modern design patterns 
- Serverless computing and cloud-native applications  
- Developing backends with Node.js, TypeScript, and Java
- Implementing asynchronous systems, RESTful APIs, and message queues
- Applying automated testing practices (unit, integration, E2E)
- Creating CI/CD pipelines and improving DevOps workflows
- Monitoring systems with observability, metrics, and performance tuning
- Participating in tech communities
